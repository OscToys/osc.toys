# Getting Started

## 1. Setup your VRChat avatar

* Most new intimate avatars and addons already include SPS.
  * If your avatar already has SPS parts, you don't need to do anything!
  * If your avatar has old DPS parts, you can upgrade [DPS to SPS](https://vrcfury.com/sps/upgrade).
  * If your avatar has nothing, you'll need to either [add sockets yourself](https://vrcfury.com/sps/#how-to-set-up-a-socket) or purchase an avatar base or asset with SPS functionality.
* **Your partners will also need SPS in order to trigger your haptics.**
  * Some worlds may also have SPS assets which can be interacted with.
  * If your partners do not have SPS, and do not upgrade their avatar, they still interact with their hands, feet, or head.
* Don't have an avatar to test with yet? Here's a public avatar with a SFW plug in one hand and SFW socket in the other: [Test Avatar](https://vrchat.com/home/avatar/avtr_892aa406-f2dd-4eab-a779-8e1222ea9d9f)

## 2. Run OscGoesBrrr

* [Download and run OGB](https://osc.toys/download)

> Using mac? OGB hasn't been tested on it, but if you are technical, you can clone the repo, make sure pnpm is installed, then `pnpm install && pnpm start`

> Using linux? We can't provide support for it, but some linux packages are available on the [github releases page](https://github.com/OscToys/OscGoesBrrr/releases).

## 3. Run Intiface

* Download Intiface Central for [Android](https://play.google.com/store/apps/details?id=com.nonpolynomial.intiface_central) or [iOS](https://apps.apple.com/us/app/intiface-central/id6444728067) or [PC](https://intiface.com/central/)
  * **Android or iOS is recommended!** Bluetooth signal on PC is usually worse or broken.
* If using the mobile app, ensure your phone is on the same wifi as your computer
* In the Intiface app:
  * Click the `App Modes` Tab
  * Enable `Show Advanced/Experimental Settings`
  * Enable `Broadcast Server Info via mDNS`
* Ensure that OscGoesBrr shows that intiface is connected, there should be a yellow or green dot
  * If mDNS doesn't work, you can try entering the server address from the intiface app into OGB's Intiface Address box manually
  * If manually enterring the address also doesn't work, make sure your phone is on the same network as your computer. If it is, your network may not allow devices to connect directly. In this case, you would be forced to use Intiface for PC.

## 4. Configure VRChat
In the main VRChat settings menu, use the search box on the top right to find and set these options:
* OSC: `On`
* Pause Avatar Interactions: `Off`
* Avatar Allowed to Interact: `Everyone`
* Avatar Self Interact: `On`
* Pixel Light Count: `High`
* Logging: `Full`

In the Shield Levels make sure the following is set:
* Lights & Particles: `On`

## 5. Link your devices

* Make sure the Intiface and VRChat sections in OGB are green (click them and check the messages if not!)
* Turn on the device, and Intiface should connect to it automatically
* When OGB detects connected devices, use the rows in the main settings list to link each device
* The default SPS links are enough for most setups, so you usually do not need to configure anything else to get started

## 6. You're done!

When your contacts are active, and are collided with by contacts of the other type, OscGoesBrrr should show an increase in levels, and your device should activate. If you need help, check out [the discord](https://osc.toys/discord)! For advanced configuration (binding certain devices to certain objects, adjusting intensity, etc) check out [this page](/settings).
